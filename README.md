# 🎬 UGC Ads – AI Ad Generator

**UGC Ads** is a full-stack AI-powered platform that allows users to generate high-quality advertisement images and videos using product and model inputs. It also features a community sharing system and credit-based billing.

## 🚀 Features

- ✅ AI-powered ad image generation using product & model inputs
- ✅ Video ad generation with a single click
- ✅ Community section to share and explore ads
- ✅ Secure authentication & credit-based billing via Clerk
- ✅ Cloud-based media storage with Cloudinary
- ✅ Fully responsive UI with smooth animations

## 📦 Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/CoderGuyShubham/UGC-Ads.git
    cd Imagify
    ```
2.  **Setup the server**
    ```bash
    cd server
    npm install
    ```
    Then run the server:
    ```bash
    npm start
    ```
3.  **Setup the client**
    ```bash
    cd ../client
    npm install
    ```
    Then run the client:
    ```bash
    npm run dev
    ```

## 📌 Usage

1. Open the application in your browser.
2. Sign up or log in using Clerk authentication.
3. Upload a product image and a model image.
4. Generate AI-based advertisement images.
5. Click to generate a video ad if required.
6. Share your generated ads in the community section.
7. Purchase credits to continue generating content.
